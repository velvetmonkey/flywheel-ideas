# @velvetmonkey/flywheel-ideas-core

Core domain logic for [flywheel-ideas](https://github.com/velvetmonkey/flywheel-ideas) — the local-first falsifiable decision ledger.

**Not intended for direct use.** Consumed by [`@velvetmonkey/flywheel-ideas`](https://www.npmjs.com/package/@velvetmonkey/flywheel-ideas) (the MCP server). Published as a separate package to keep the dependency chain honest across the workspace.

Ships the domain types, SQLite schema + migrations, write helpers, lifecycle state machine, assumption + council + outcome logic, and the CLI error classifier.

Full docs + install instructions live at the [main repo README](https://github.com/velvetmonkey/flywheel-ideas#readme).
